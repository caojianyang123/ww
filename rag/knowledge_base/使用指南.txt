智能助手使用指南

一、系统要求
1. 操作系统：Windows 10/11 或 Linux
2. Python版本：3.9及以上
3. 内存要求：至少8GB
4. 网络要求：需要联网访问阿里云百炼API

二、安装步骤
1. 下载项目代码
2. 安装依赖：pip install -r requirements.txt
3. 配置环境变量：设置DASHSCOPE_API_KEY
4. 启动服务：python main.py

三、上传知识库
1. 启动Streamlit服务：streamlit run rag/app.py
2. 在浏览器中打开页面
3. 选择文档文件（支持txt、pdf）
4. 点击上传并向量化按钮
5. 等待向量化完成

四、问答操作
1. 在终端运行 python main.py
2. 输入问题并回车
3. 系统会根据知识库内容进行回答
4. 输入"退出"或"quit"结束对话

五、注意事项
1. 首次使用需要上传知识库文档
2. 文档大小建议不超过10MB
3. 支持的文档格式：txt、pdf
4. 上传新文档后建议重建索引
5. API密钥需要妥善保管，不要泄露

六、常见问题
Q: 无法连接到阿里云API？
A: 请检查网络连接和API密钥是否正确

Q: 回答不准确？
A: 请检查知识库是否包含相关内容，或尝试增加训练数据

Q: 如何提高回答速度？
A: 可以减少TOP_K参数，或使用更强大的硬件资源
